# Hotel Management JavaFX Application

A JavaFX-based application for managing hotels and rooms. Users can create hotels, add/remove regular and deluxe rooms, search for hotels, and display all hotels.

## Features
- Create and manage hotels
- Add/remove regular and deluxe rooms
- Search for hotels by name
- Display all hotels
- Reset functionality

## How to Use
1. **Create a Hotel**: Enter a name and click "Create Hotel."
2. **Search for a Hotel**: Enter the hotel name and click "Search Hotel."
3. **Add a Room**: Select a hotel, enter room details, and click "Add Regular Room" or "Add Deluxe Room."
4. **Remove a Room**: Select a hotel, enter the room number, and click "Remove Room."
5. **View All Hotels**: Click "Show All."
6. **Reset**: Clear the hotel list or display area with the "Reset" buttons.

## Requirements
- Java 8 or higher
- JavaFX SDK
